## Overview

The Purchase Orders module is the central hub for creating and managing purchase orders. From the main page, you can search POs, create new orders, and navigate to Open POs, Back Ordered Part Requests, and Restock for inventory replenishment.

## What You Can Do

- [Purchase Order Detail](/docs/purchase-orders/purchase-order-detail) — Create and edit purchase orders (parts, equipment, misc)
- [Open Purchase Orders](/docs/purchase-orders/open) — View and manage open POs by branch and department
- [Back Ordered Part Requests](/docs/purchase-orders/back-orders) — Select parts from work orders to add to a new PO
- [Restock](/docs/purchase-orders/restock) — Parts to order by branch for inventory replenishment

## Main Page Features

### Search

Use the **Search Purchase Orders** combobox to find POs by PO number or vendor name. Select a result to open the purchase order detail page.

### Quick Actions

- **Open Purchase Orders** — View open POs filtered by branch and department
- **Back Ordered Part Requests** — Parts from work orders that are back ordered; select and create a PO
- **Restock** — Parts needing restock by branch; select part groups and create a PO

### New Purchase Order

Click **New Purchase Order** to create a new PO. You can also create a PO from Back Orders or Restock by selecting parts and clicking **Create Parts Order** or **Create Order** — they navigate to the new PO form with the selected parts pre-filled.
