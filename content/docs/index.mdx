## Welcome

Welcome to the NOVA Manager documentation. This site explains how to use the application day to day, and also includes background on the architecture and the migration from the legacy system.

- If you are a **service manager or dispatcher**, start with [Work Orders](/docs/work-orders).
- If you are a **parts manager**, start with [Parts Manager](/docs/parts).
- If you work in **accounting or finance**, start with [Accounting](/docs/accounting).
- If you are **new to the project**, start with [Project Overview](/docs/project-overview).
