## Overview

The Labor Manager module is the central hub for labor approval, timecard documents, and labor KPIs. From the main page, you can navigate to Approval (import pending labor to work orders), Timecards (view signed timecard PDFs by technician), and the KPI Dashboard (key metrics for labor managers).

## What You Can Do

- [Labor Approval](/docs/labor/approval) — Review and import technician labor by tech; approve pending entries
- [Timecards](/docs/labor/timecards) — View signed timecard PDFs grouped by technician; export ZIP
- [KPI Dashboard](/docs/labor/kpi) — Pending approval, open labor value, hours billed, utilization, overtime, aging

## Main Page

The Labor Manager main page displays **Quick Actions** — cards that link to each submodule. Click any card to open that area.
