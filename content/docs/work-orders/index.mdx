## Overview

The Work Orders module is the central hub for managing service work orders, quotes, and rentals. From the main page, you can search work orders, access recently viewed orders, and navigate to specialized areas for creation, recurring processing, closing, and reporting.

## What You Can Do

- [Create Work Order](/docs/work-orders/create) — Create new work orders or quotes
- [Recurring Orders](/docs/work-orders/recurring) — Close and open recurring orders (GM/Rental, PM)
- [Close / Distribute](/docs/work-orders/close-distribute) — Centralized closing and distribution
- [Credit Approval](/docs/work-orders/credit-approval) — Approve pending credit items
- [Workflow](/docs/work-orders/workflow) — Approve work items to send to customer
- [Reports](/docs/work-orders/reports) — Interactive reports dashboard

## Main Page Features

### Search

- **Quick search** — Work order search combobox for fast lookup by WO number, customer, or equipment
- **Advanced search** — Filter by type (orders vs quotes), branch/dept, Ship To, Bill To, Serial #, GM Serial #, Unit #, PO #

### Recently Viewed

The main page displays the last 10 work orders you've viewed, with status badges and equipment info. Click any card to open the work order detail.

### Work Order Statuses

| Status | Description |
|--------|-------------|
| Open | Active work order |
| Accepted | Quote accepted by customer |
| Quote | Pending quote |
| Closed | Completed and closed |
| Rejected | Quote rejected |
| Voided | Voided order |

## Work Order Detail

Click any work order to open its detail page. See [Work Order Detail Page](/docs/work-orders/work-order-detail) for an overview of the layout, tabs, and key actions.
