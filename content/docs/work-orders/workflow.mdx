## Overview

Workflow is used to approve quote line items before they are sent to the customer. Quotes with paperwork complete appear here. Each quote has workflow types (such as Parts, Labor, Equipment, Rental) that must be approved before the quote can go out.

## Workflow Types

Workflow types are configured for your organization. Common types include:

- **Parts** — Parts line items
- **Labor** — Labor line items
- **Equipment** — Equipment line items
- **Rental** — Rental line items

Each type appears as a column in the table. You approve each type separately for each quote.

## Workflow

1. Select a **Branch** and **Department** to filter quotes
2. Review the list of quotes pending workflow approval
3. For each workflow type, click **Approve** to approve that section
4. Once approved, the checkmark appears and you can click **Record** to record the approval with date
5. Click the quote number to open the work order detail
6. Use **Refresh** to reload the list

## Quote Table

| Column | Description |
|--------|-------------|
| Quote # | Quote/work order number (click to open) |
| Customer | Ship-to customer |
| Serial | Equipment serial number |
| Comments | Quote notes |
| [Workflow Type] | One column per type (Parts, Labor, etc.) |

## Approval Status

- **Pending** — Circle icon; click **Approve** to approve
- **Approved** — Checkmark icon; shows who approved; click **Record** to record the approval

## After Approval

When all required workflow types are approved for a quote, it can be sent to the customer. The approval is tracked with the approver's name and date.
